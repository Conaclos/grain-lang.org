Grain

a modern web staple

Grain is a language built for the modern web by leveraging the brilliant work done by the WebAssembly project. Discover new ways to build web applications— and do so in a fun and exciting functional way.

Language Features

Sit back and enjoy the bountiful harvest of Grain's rich feature set.

Bindings 


  let x = 5 in
  let y = 6 in
    x + y

These work just as you'd expect of any reasonable language.

Functions 


  let f = (lambda z: (lambda x, y: z + x + y)) in
    f(5)(4, 3)

Lambdas are king (and by that we mean first-class) in Grain. Here we show the variable 'z' being closed over inside the inner lambda.

Recursion 


  let rec is_even = lambda n:
    if (n <= 1):
      n == 0
    else:
      is_odd(n - 1)
  and is_odd = lambda n:
    if (n <= 1):
      n == 1
    else:
      is_even(n - 1)
  in
    is_odd(999)

No functional language is complete without recursion.

Tuples & Mutation 


  let tup = (0, 0) in
    begin
      tup[0] := 5;
      tup[1] := 4;
      if tup[1] > 3:
        tup[0] := tup[0] - 3
      else:
        tup[0] := tup[0] + 3;
      tup[1] - tup[0]
    end

If that's your kind of thing.

DOM Manipulation 


  let div1 = DOM::query("#div1") in
  let div2 = DOM::query("#div2") in
  let div3 = DOM::query("#div3") in
    if equal(div1, false) || equal(div2, false) || equal(div3, false):
      print("An element was not found!")
    else:
      begin
        DOM::setText(div1, "I'm just a div.");
        DOM::setText(div2, "Sitting here on wasm hill.");
        DOM::dangerouslySetInnerHTML(div3, "
[[grain]]
");
      end

DOM nodes are primitives in Grain. Traditional DOM queries and node operations are available.

DOM Interaction 


  let counter = (0,) in
  let button = DOM::query("#button1") in
  let view = DOM::query("#div1") in
  let handleEvent = (lambda:
    begin
      counter[0] := counter[0] + 1;
      DOM::setText(view, toString(counter[0]))
    end) in
    DOM::addEventListener(button, "click", handleEvent)

Native Grain functions can be bound as callbacks to native DOM events. This example updates a counter whenever the button is clicked.
